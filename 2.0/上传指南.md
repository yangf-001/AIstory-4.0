# Git 快速上传方法

## 方式一：使用 PowerShell 脚本（推荐）

创建一个 `upload.ps1` 文件：

```powershell
# 上传配置
$repoPath = "c:\Users\yf217\Desktop\aichat\开发\开发2.0（故事生成器）\3.2.0"
$commitMessage = "更新代码"

Set-Location $repoPath
git add .
git commit -m $commitMessage
git push origin master
```

运行命令：
```powershell
powershell -ExecutionPolicy Bypass -File upload.ps1
```

---

## 方式二：使用命令行

```powershell
# 1. 进入项目目录
Set-Location "c:\Users\yf217\Desktop\aichat\开发\开发2.0（故事生成器）\3.2.0"

# 2. 添加所有文件
git add .

# 3. 提交（使用英文消息避免编码问题）
git commit -m "Update"

# 4. 推送到 GitHub
git push origin master
```

---

## 方式三：VS Code 图形化操作

1. 打开 VS Code
2. 点击左侧 Source Control 图标
3. 输入提交信息，点击 ✓ 提交
4. 点击 🔄 同步更改

## 首次上传配置（仅执行一次）

```powershell

git init

# 配置用户信息
git config user.email "2178099763@qq.com"
git config user.name "yangfan"

# 添加远程仓库
git remote add origin https://github.com/yangf-001/AI-stoy3.2.0.git
```

## 常用命令

| 操作 | 命令 |
|------|------|
| 查看状态 | `git status` |
| 查看提交历史 | `git log` |
| 拉取最新代码 | `git pull origin master` |
| 强制推送 | `git push -f origin master` |

---

## 注意事项

- 中文路径在某些环境下可能有问题，建议使用英文路径或 PowerShell 脚本
- 首次使用需要在 GitHub 上创建仓库
- 提交信息建议使用英文
